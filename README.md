# Query Theory

#### Interactive, Cross-Platform Quiz and Study Application

Query Theory is a dynamic educational app inspired by Quizlet, designed to enhance user knowledge retention through varied study modes, timed testing, and seamless content management. It transforms static study material into engaging, real-time learning sessions.

## ‚ú® Core Features

* **Timed Quiz Mode:** Integrated real-time **time clock** to challenge users and simulate test pressure, recording performance metrics upon completion.

* **Diverse Study Sections:** Supports multiple learning styles through distinct modes:
    * **Quiz Mode:** Standard question-and-answer format with score tracking.
    * **Study Mode:** Allows users to flip through cards without pressure.
    * **Review Mode:** Focuses specifically on previously answered incorrect questions.

* **Content Creation:** Users can easily create and manage their own custom quiz sets and study cards.

## üöÄ Architectural Features & Scalability

* **Offline Access:** Supports syncing and storage of quiz data for seamless use without a continuous internet connection.

* **Progress Tracking:** Tracks detailed user history, including scores, time taken per quiz, and knowledge gaps (incorrect answers).

* **Intuitive UI/UX:** Designed with a clean, responsive interface for easy navigation between study sections and modes.

* **Secure User Accounts:** Features secure user sign-up and login for personalized progress saving.
